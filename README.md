# we-buy
A bulk purchase web-app built using the MERN stack

## How to run

```(shell)
$ cd path/to/directory
```

Install required packages
```(shell)
$ cd frontend
$ npm i
$ cd ../backend
npm i
```

Run the frontend and backend servers
```(shell)
$ cd frontend
$ npm start
$ cd ../backend
$ npm run server
```

Visit `http://localhost:3000` in your browser.
